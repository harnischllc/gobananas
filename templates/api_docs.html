Programmatic access to banana ripeness classification

POST /api/classify

Classify banana ripeness via image upload or hex color.

Request
  • Multipart form: field image (PNG, JPG, JPEG, GIF, BMP; max 10MB)
  • JSON: { "color": "#FFD700" } or "FFD700"
Response (200)
{
  "stage": 6,
  "stage_info": "Yellow - Peak eating quality. Aromatic and sweet.",
  "days_until_peak": 0,
  "hue": 58.3,
  "success": true
}
Errors
  • 400: invalid input (file type/size, bad hex)
  • 500: server error during analysis
